#### Example - indeterminate {#ExampleProgressIndeterminate}

Width, speed, timing function and direction of indeterminate progress bar can be customized with css variables:

- `--progress-bar-timing`
- `--progress-bar-speed`
- `--progress-bar-width`
- `--progress-bar-direction`
