# Vue-wind

UI components collection for Vue 3 applications. 

Vue-wind uses **Tailwindc CSS** as CSS framework and **Floating UI** for positioning.

```
npm -i vue-wind @floating-ui/dom
```

[Documentation and examples](https://vue-wind.netlify.app/documentation/installation)

Components:

* Alert
* Badge
* Breadcrumb
* Button
* Card
* Chevron
* Close button
* Collapse (accordion)
* Date Picker
* Divider
* Dropdown (context menu)
* Icon
* List
* Modal
* Navbar
* Pagination
* Popover
* Progress
* Spinner
* Sidepanel
* Tabs
* Table
* Tooltip (directive)
* Tree

Form components:

* Autocomplete
* Checkbox (checkbox group)
* Form
* Form Text
* Input
* Radio (radio group)
* Range
* Select
* Textarea
